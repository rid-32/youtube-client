## Installation

In the project directory run:

### 'yarn'

or

### 'npm install'

## Development mode

In the project directory run:

### `yarn dev`

or

### `npm run dev`

## Production mode

In the project directory run:

### `yarn build`

or

### `npm run build`

After that you'll get the "build" folder.

## Build production bundle and run it:

### `yarn start`

or

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
